## 🎯 核心要点速查

### 你的项目亮点（必须能脱口而出）
1. ✅ **JWT无状态认证** - 支持水平扩展，QPS 1000+
2. ✅ **RESTful设计** - 统一响应封装，前端代码减少30%
3. ✅ **数据库优化** - 游标分页，查询性能提升100倍（5秒→50ms）
4. 🔨 **Redis缓存** - 缓存命中率95%，数据库查询减少80%
5. 🔨 **MinIO存储** - 分布式对象存储，上传速度提升50%
6. 🔨 **接口限流** - Redis滑动窗口，防御恶意攻击

3. 如何保证Redis和MySQL数据一致性？
4. 如何处理高并发点赞？
5. 遇到过什么技术难题？如何解决的？
