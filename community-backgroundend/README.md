# 社区技术论坛 (Community Tech Forum)

## 项目描述

这是一个用于学习和实践现代Java全栈技术的Web应用项目。目标是构建一个功能虽简但结构完整、代码规范的社区论坛，并以此项目作为求职的个人作品。

## 核心功能列表 (V1.0)

- **用户注册**: 作为一个新访客，我希望能注册一个账号，以便我能发帖和评论。
- **用户登录**: 作为一个已注册用户，我希望能登录我的账号，以便系统能认识我。
- **发布帖子**: 作为一个已登录用户，我希望能发布新帖子，以便分享我的想法。
- **浏览帖子**: 作为任何用户，我希望能看到所有帖子的列表，以便我能发现感兴趣的内容。
- **查看详情**: 作为任何用户，我希望能点击查看单个帖子的详情。

## 技术选型草案

- **后端**: Spring Boot, Spring Security, MyBatis, MySQL
- **前端**: Vue.js
- **构建工具**: Maven
